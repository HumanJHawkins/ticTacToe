Help!

Tic-tac-toe is a traditional usually children's game where players take turns placing a mark in a 3 x 3 grid until one of them has three in a row, or until no squares are left.

Guesses can be made by pressing keys on the keyboard while the game window is active, or by selecting (clicking or touching) the square of the tic-tac-toe board where a mark should be placed. Difficulty settings affect how likely the computer is to make a poor choice.

Good luck!

About Tic-Tac-Toe

This implementation of Tic-tac-toe is intended to illustrate the relatively new CSS Grid module (supported by major browsers beginning late in 2017). If you are reading this in 2025, I suppose it's not really new anymore.

Full source code is available at https://github.com/HumanJHawkins/tic-tac-toe, with the most interesting bits (to me anyway) in the .css file there.
